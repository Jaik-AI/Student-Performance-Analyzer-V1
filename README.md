# Student-Performance-Analyzer-V1
A Python-based student performance analyzer that calculates average marks, identifies toppers, and analyzes student records using file handling.

## Features
- Add student records
- View student records
- Calculate average marks
- Find topper
- Find lowest scorer
- Input validation

## Technologies Used
- Python
- File I/O
- Functions
- Exception Handling

## How to Run
python main.py

## What I Learned
- Working with structured data
- File processing
- Average calculations
- Finding maximum and minimum values
- Input validation
